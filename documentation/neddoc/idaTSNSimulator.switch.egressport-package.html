Package idaTSNSimulator.switch.egressport

Name Type Description
Arbiter module interface

The Arbiter Module is a modulinterface for different Arbiter and defines In- and Oututs.
ArbiterAVB simple module

The ArbiterAVB is a module which is conected to Arbiter module interface The ArbiterAVB arbitrate the in BuffersModule stored Frames and send them out Based on the Credit Based Shaper mechanism the frames are selected
ArbiterIEEE802_1Q simple module

The ArbiterIEEE802_1Q is a module which is conected to Arbiter module interface The ArbiterIEEE802_1Q arbitrate the in BuffersModule stored Frames and send them out Based on priorites the frames are selected
ArbiterIEEE802_1Qbv simple module

The ArbiterIEEE802_1Qbv is a module which is conected to Arbiter module interface The ArbiterIEEE802_1Qbv arbitrate the in BuffersModule stored Frames and send them out Frames are selected for transmission based on the IEEE 802_1Qbv.
ArbiterIEEE802_3 simple module

The ArbiterIEEE802_3 is a module which is conected to Arbiter module interface The ArbiterIEEE802_3 arbitrate the in BuffersModule stored Frames and send them out Based on round robin principle the frames are selected
BuffersModule simple module

BuffersModule manages the buffers per port and includes an array of FrameBuffers (one Frame Buffer per individual traffic class / priority)
Classifier module interface

The Classifier Module is a modulinterface for different Classifiers.
ClassifierPriority simple module

The ClassifierPriority is a Classifier based on priority filtering and is an interface for the Classifier Module.
EgressPort compound module

The EgressPort is a module which include the classifier, buffersmoudle and the arbiter module to process and store forwarded frames based on the selected classifier and arbiter the behaviour can be influenced

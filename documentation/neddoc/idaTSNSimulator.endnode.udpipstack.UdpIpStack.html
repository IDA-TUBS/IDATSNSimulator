Simple Module UdpIpStack

Package: idaTSNSimulator.endnode.udpipstack
File: src/endnode/udpipstack/UdpIpStack.ned

C++ definition

Passes the data from the middleware and pure Ethernet generators to the ethernet network ports. Passes the data from the Ethernet network ports to either the middleware or the pure Ethernet sink.

Used in compound modules:

Name Type Description
EndNode compound module

The endnode is the high-level module for conncetion to the network. Therefore it enables to send or receive Ethernet or RTPS message streams via dedicated Ethernet ports.

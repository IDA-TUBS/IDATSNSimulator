Detailed Description

Time aware shaper (IEEE802.1Qbv) is a circuit-switched arbitration. To evaluate the arbiter the TimeAwareShaperSegmentStatistics draws the four time segments:

  1. Critical sending, 2. critical guardband, 3. noncritical sending,
  2. noncritical guardband.
